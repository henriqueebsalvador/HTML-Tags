Tags de HTML

Tag de Comentário

A tag de comentário serve para escrever algo que não aparece na página.

Exemplo de código:

<!-- Isso é um comentário -->

O comentário não aparece na tela.

Tags h1 até h6

São usadas para títulos. h1 é o maior e h6 é o menor.

Exemplo:

<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>
<h5>Título 5</h5>
<h6>Título 6</h6>

Tag P (Parágrafo)

Serve para escrever um parágrafo de texto.

Exemplo de código:

<p>Esse é um parágrafo.</p>

Esse é um parágrafo de exemplo.
